# Plugin images (optional)

Place your branding images here. All are optional; the plugin uses WordPress dashicons if custom files are missing.

| File       | Purpose | Recommended size |
|-----------|---------|-------------------|
| **icon.png** or **icon.svg** | Admin menu icon (left sidebar) | 20×20 px or 24×24 px; SVG any size |
| **logo.png** or **logo.svg** | Logo on admin page headers (Connections, Mappings, Logs) | e.g. 200×50 px; SVG scales |

- **Menu icon:** Name the file `icon.png` or `icon.svg`. If present, it replaces the default database icon in the WordPress admin menu.
- **Header logo:** Name the file `logo.png` or `logo.svg`. If present, it appears below the page title on each CF7 Database Connector admin page.

After adding files, refresh the admin to see them.
